## ⚡ Установка

1. Скачай [последний релиз](https://github.com/Exitze/ZIPLINES-releases/releases/latest)
2. Открой `ZIPLINES-X.X.X.zip`
3. Перетащи `ZIPLINES.app` в папку **Приложения**
4. Запусти — готово!

> ZIPLINES автоматически уведомит тебя о новых обновлениях.

## ✨ Возможности

- **QuickLook** — просматривай содержимое архивов прямо в Finder (пробел)
- **Share Extension** — сжимай файлы из любого приложения через кнопку «Поделиться»
- **Spotlight** — ищи файлы внутри архивов через Spotlight
- **AES-256 + Touch ID** — шифрование и сохранение паролей в Keychain
- **Split volumes** — разбивка архивов на части любого размера
- **Автообновление** — уведомления о новых версиях прямо в приложении
